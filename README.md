# Avaliador de Desligamento - Versão Web

Este projeto foi otimizado para rodar diretamente no navegador e ser hospedado online.

## 🚀 Como colocar online (Grátis no Vercel)

Como você não possui o Git instalado, siga estes passos simples:

1.  Acesse [vercel.com](https://vercel.com/) e crie uma conta (pode usar seu e-mail).
2.  No seu computador, selecione todos os arquivos desta pasta **EXCETO** `node_modules`, `release` e `.git`.
3.  Crie um arquivo `.zip` com esses arquivos.
4.  No painel do Vercel, clique em **"Add New"** -> **"Project"**.
5.  Em vez de importar do Git, procure pela opção de **"Upload"** ou arraste e solte o seu arquivo `.zip` na área indicada.
6.  O Vercel irá processar e te dar um link oficial em segundos!

## 📱 Vantagens
- Funciona em qualquer celular.
- Pode ser "instalado" no celular (Adicionar à tela de início).
- Impressão de PDF integrada.

## 🛠️ Comandos Locais
- `npm install`: Instala as dependências.
- `npm run dev`: Roda o app localmente para testes.
- `npm run build`: Gera a versão final para subir no site.
